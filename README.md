# Bombmanual

A automatic program for game "Keep Talking and Nobody Explodes".

Assistance for modules that are easy to communicate but take a long time to compute.

# Features

- [ ] Read static bomb information
  - [x] battery
  - [x] serial number
  - [ ] ports
  - [ ] indicators
- [x] Wire module
- [ ] Button module
- [ ] Complex wire module
- [ ] Morse code module
- [ ] Password module
